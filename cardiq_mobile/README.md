## Build & Run Commands

Run all commands from the `cardiq_mobile` directory:

### 1. Android Release APK Build
```bash
flutter build apk --release --dart-define-from-file=../cardiq/.env
```

### 2. Local Testing & Development (with Hot Reload)
```bash
flutter run --dart-define-from-file=../cardiq/.env
```

### 3. iOS Release Build (IPA Archive)
```bash
flutter build ipa --release --dart-define-from-file=../cardiq/.env
```
