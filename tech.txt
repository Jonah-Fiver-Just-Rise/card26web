1. Firebase Credentials
-Go to the Firebase Console and sign in. https://console.firebase.google.com/
-Click Add project (or select your existing project if you already made one).
-Once the project dashboard loads, click the Web icon </> (or go to Project Settings (gear icon) > General and scroll down to Your apps to add a web app).
- Register your app with a nickname (e.g., cardiq-web).
-Firebase will show you an initialization code snippet containing the keys:

const firebaseConfig = {
  apiKey: "...",        // VITE_FIREBASE_API_KEY
  authDomain: "...",    // VITE_FIREBASE_AUTH_DOMAIN
  projectId: "...",     // VITE_FIREBASE_PROJECT_ID
  storageBucket: "...", // VITE_FIREBASE_STORAGE_BUCKET
  messagingSenderId: "...", // VITE_FIREBASE_MESSAGING_SENDER_ID
  appId: "..."          // VITE_FIREBASE_APP_ID
};
- Copy these values directly into the matching variables in your .env file.
- Go to the left-hand menu in Firebase, click Build > Authentication, click Get Started, and enable the Email/Password sign-in provider.

2. OpenAI (ChatGPT) API Key 
-Go to the OpenAI Developer Platform and sign in. https://platform.openai.com/
-Under the left-hand navigation pane, click API Keys.
-Click the Create new secret key button.
-Give it a name (e.g., CardIQ Development), click create, and copy the key immediately (you won't be able to see it again).

For Android:
Go to your Firebase Console.
Go to Project Settings (gear icon) > General.
Under Your apps, click Add app and select the Android platform icon.
Set the package name to com.justrise.cardiq.cardiq_mobile (matching your organization package).
Download the google-services.json file.
Place this file inside: cardiq_mobile/android/app/google-services.json

For iOS:
Under Your apps in the Firebase Console, click Add app and select the iOS platform icon.
Enter the Bundle ID com.justrise.cardiq.cardiqMobile.
Download the GoogleService-Info.plist file.
Place this file inside: cardiq_mobile/ios/Runner/GoogleService-Info.plist (and open the iOS folder in Xcode to drag and link this file into the Runner folder).
